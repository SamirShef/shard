The escape sequence must be supported.

Escape sequences:
1) `\a`
2) `\b`
3) `\e`
4) `\f`
5) `\r`
6) `\n`
7) `\t`
8) `\v`
9) `\\`
10) `\'`
11) `\"`
12) `\?`
13) `\onnn` (`nnn` is an octal number consisting of three digits)
14) `\xhh` (`hh` is a hexadecimal number consisting of two digits)

Bad literal:
```shard
'\z'
```

Good literal:
```shard
'\o123' // `S` letter
```
