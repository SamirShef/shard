The object is not a structure.

Bad code:
```shard
fun main() {
    var max_i32: i32 = i32.max;
//                     ^^^ is not a structure
}
```

Good code:
```shard
struct I32 {
    pub var max: i32
}

fun main() {
    var int: I32 = I32 { max: 2147483647 };
    var max_i32: i32 = int.max;
}
```
