Unsupported member of the structure.

Bad code:
```shard
struct Human {
    pub age: i32,
    fun get_age() : i32 {
        return age;
    }
//  ^^^^^^^^^^^^^^^ structure has only fields
}
```

Good code:
```shard
struct Human {
    pub age: i32
}

impl Human {
    pub fun get_age() : i32 {
        return this.age;
    }
}
```
